# DelphiAnalysisMcpServer - No Changes

## Status

This project has **no changes** in this version. It is included for completeness.

The MCP Server continues to provide:
- SQL query extraction from Delphi source code
- Integration with DelphiDocumenter
- Tools for code analysis

## Purpose

The MCP Server extracts SQL queries from Delphi code files and helps link them to methods. It works in conjunction with DelphiDocumenter to provide comprehensive code analysis.

## Building

```bash
npm install
npm run build
```

## Usage

Run as an MCP server for Claude Desktop or other MCP clients.

See the main README.md for full usage instructions.
